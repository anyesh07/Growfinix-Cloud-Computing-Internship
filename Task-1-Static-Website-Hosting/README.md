# Task 1: Static Website Hosting on AWS S3

## Objective

The objective of this task is to deploy and host a static website using Amazon S3.

## AWS Services Used

- Amazon S3

## Steps Performed

1. Created an Amazon S3 bucket.
2. Uploaded the static website files to the S3 bucket.
3. Enabled Static Website Hosting.
4. Configured the index document as `index.html`.
5. Disabled Block Public Access for the bucket.
6. Added a bucket policy to allow public read access.
7. Accessed and verified the deployed website using the S3 website endpoint.

## Result

The static website was successfully deployed and hosted using Amazon S3.

## Website Endpoint

`http://growfinix-task1-anyesh.s3-website-us-east-1.amazonaws.com`
